# blogify

A backend‑focused blogging platform built with Node.js, Express, EJS templates, and MongoDB (via Mongoose).
Users can sign up, log in, write and publish blog posts, and add comments. Everyone (including unregistered visitors) can browse posts and view discussions.

## 🚀 Features

- Register and sign in using secure authentication with cookies.
- Publish blog posts after signing in.
- Browse all published articles without an account.
- Leave comments on posts after signing in.
- View comment threads as any visitor.
- See and manage your own posts and comments.

## 🛠 Technology Used

- **Node.js & Express** – Backend server and routing.
- **EJS** – Server‑rendered HTML templates.
- **MongoDB + Mongoose** – Database and object modeling.
- **cookie‑parser** – Cookie handling for user sessions.
- **Bootstrap** – Basic UI styling.

## 🚀 Installation & Setup

1. **Clone the repository**

    ```bash
    git clone https://github.com/letgit0/blogify.git
    cd blogify
    ```

2. **Install packages**

    ```bash
    npm install
    ```

3. **Create a .env file**

    Add your environment variables.

4. **Start the server**

    ```bash
    npm start
    ```

5. **Open in browser**

    ```
    http://localhost:3000
    ```
